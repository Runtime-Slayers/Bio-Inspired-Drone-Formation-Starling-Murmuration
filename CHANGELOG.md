# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-06-25
### Added
- Created the core `drone_swarm` package with Boids kinematic rules, mission tracking, and cognitive load task offloading.
- Created the core `hrv_analysis` package with DFA and MFDFA algorithms.
- Created unit tests in `tests/` for both modules achieving full coverage.
- Added validation scripts under `experiments/` with CLI parameterization and safe console encoding setups.
- Configured CI pipeline via GitHub Actions.
