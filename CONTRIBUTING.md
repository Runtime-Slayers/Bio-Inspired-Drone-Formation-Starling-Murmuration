# Contributing to Swarm Drone & HRV MFDFA Repo

We welcome community contributions! Please adhere to the following guidelines:

## Code Guidelines
- Write clean, Pythonic code.
- Ensure type hints are used for all public APIs.
- Write unit tests for all new features or bug fixes.
- Follow the PEP 8 style guide.

## Pull Request Process
1. Fork the repository and create a new branch.
2. Verify all tests pass locally by running `pytest`.
3. Submit a pull request detailing your changes.
4. The CI pipeline must run successfully and pass all code validation gates before merging.
